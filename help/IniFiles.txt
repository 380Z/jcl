<DECL>--------------------------------------------------------------------------
[Declaration]
procedure IniWriteBool(const FileName, Section, Line: string; Value: Boolean);
[ShortDescr]
Writes a boolean key to an ini file.
[Description]
IniWriteBool writes a single boolean key to the specified section in the
specified ini file. If the function fails it raises an exception.
[Parameters]
Count=3
1=FileName:string=The fully qualified filename of the ini file.
2=Section:string=The section in the ini file to which to write a value
3=Line:string=The name of the key to write
4=Value:Boolean=The value to write
[Quick Info]
BCB=3,4,5
Unit=JclIniFiles
Category=Registry and Ini files
SubCat=Ini files
Author=John C Molyneux;jaymol@hotmail.com
Delphi=3,4,5
Windows NT/2000=Requires Windows NT 3.1 or later.
Windows 95/98=Requires Windows 95 or later.
[SeeAlso]
Count=5
1=IniReadBool
2=IniReadInteger
3=IniReadString
4=IniWriteInteger
5=IniWriteString

[Declaration]
function IniReadBool(const FileName, Section, Line: string): Boolean;
[ShortDescr]
Reads a boolean key from an ini file.
[Description]
IniReadBool reads a single boolean value within the specified section from the
specified ini file. If the function fails it returns the default value False.
[Parameters]
Count=3
1=FileName:string=The fully qualified filename of the ini file.
2=Section:string=The section in the ini file from which to read a value
3=Line:string=The name of the key to read
[Result]
The key within section, default value returned on failure is False.
