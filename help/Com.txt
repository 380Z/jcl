<DECL>--------------------------------------------------------------------------
[Declaration]
function IsDCOMInstalled: Boolean;
[ShortDescr]
Returns whether DCOM is installed.
[Description]
IsDCOMInstalled returns whether DCOM is installed on the target computer. This
routine works on all versions of Windows but since DCOM is installed by default
on all operating systems except Windows 95 (in it's various incarnations) the
routine only performs an actual test on Windows 95. Note that DCOM being intalled
does not necessarilly mean it's all enabled. To determine that use the IsDCOMEnabled
routine.
[Result]
If DCOM is installed the routine returns True, otherwise it returns False.
[Quick Info]
Unit=JclCom
Category=Component Object Model
SubCat=Version Utilities
Author=Kevin S. Gallagher
Delphi=4,5,6
Kylix=1
FPC=1.x
[SeeAlso]
Count=2
1=IsDCOMEnabled
2=GetDCOMVersion
<DECL>--------------------------------------------------------------------------
[Declaration]
function IsDCOMEnabled: Boolean;
[ShortDescr]
Returns whether DCOM is enabled.
[Description]
IsDCOMEnabled returns whether or not DCOM is enabled on the target machine. Note
that enabled and installed are two different things. To test whether DCOM is
installed use the IsDCOMInstalled routine.
[Result]
If DCOM is installed the routine returns True, otherwise it returns False.
[Quick Info]

Category=Component Object Model
SubCat=Version Utilities
Author=Kevin S. Gallagher
Delphi=4,5,6
Kylix=1
FPC=1.x
[SeeAlso]
Count=1
1=IsDCOMInstalled
2=GetDCOMVersion
<DECL>--------------------------------------------------------------------------
[Declaration]
function GetDCOMVersion: string;
[ShortDescr]
Returns the version of DCOM installed.
[Description]
GetDCOMVersion returns the version of DCOM installed on the target computer.
To determine the actual version from the result use the table as provided on the
Microsoft website http://support.microsoft.com/support/kb/articles/Q235/6/38.ASP.
[Result]
Returns the DCOM version in the form a.b.c.d.
