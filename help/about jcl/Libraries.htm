3rd Party Code Libraries

The JEDI Code Library is a powerfull library but it's not the only one of it's kind. There are various other code libraries available, both freeware/shareware, open source and commercial. I highly recommend you have a look at a few of these, some of them are quite good and can be a powerfull addition your toolbox. Note that some of the free libraries are included in the JCL (or will be in the near future).

This is not an extensive list, if you know of other libraries please let us know about them so we can include them here.

Open Source/Freeware/Shareware Libraries

ESBMaths/ESBDates/ESBRoutines Various utility routines.

Mike Lischke's GraphicEx Enhancement of Graphics.pas for loading of numerous image formats.

Alex Denissov's Graphics32 Very fast 32 bit graphics routines.

EZDSL Data Structures Library.

Delphi Encryption Compendium Part I-III (Cipher) Implementation of numerous encryption algorithms.

EFD HyperString Various string routines (without sourcecode).

Delphree's Nexus Opensource application framework.

Delphi Fundamentals Various utility routines similar to the JCL.

Maps Generic associative containers library.

Numerical Methods in Pascal Homepage.
