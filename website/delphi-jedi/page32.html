JediVCS

The JEDI Code Library Team uses SVN for Version Control. To always have access to the most recent changes in the JCL, you should install a SVN client (we recommend TortoiseSVN) and download the SVN repository files to your computer by checking out the repository at http://jcl.svn.sourceforge.net/svnroot/jcl/trunk/jcl/.

With the SVN client, you can update your local repository at any time.

You can also access the SVN repository via the web. Click this link to go to the SVN trunk.
